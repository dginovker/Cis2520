Single Row Bejeweled

For this problem, you will implement a 'char' stack for which the interface will be provided. You must implement 3 methods from the stack: getTop( Stack * ) push(Stack *, char ); pop(Stack*) Bejeweled is a game where players try to connect matching colors to score points. In this version of bejeweled, when 3 or more of the same color squares are adjacently connected, the squares are removed from the board. The player is awarded 10 points for each square removed. In this version, the game board will be a single column and you'll use your stack to identify if the player can win and how many points they could score. The square colors are placed in the game board one at a time and take the lowest available position in the column. When 3 colors are stacked you'll remove them and award the player points. The game ends when no more pieces are available. At that point, the player has either won or lost the game. They win if no blocks are left on the game board, otherwise, they lose. The colors arrive in order from left to right. Input: BBB Output 30 Win Input BRRRGBB 30 Lose
